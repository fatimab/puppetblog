Installation et configuration


Installation

Pour commencer, vous devez vous assurer d'être bien connecté à internet.
Ensuite, il y a 2 méthodes selon votre système d'exploitation :

Pour un système utilisant APT

Sur le client : apt-get install puppet
Sur le serveur : apt-get install puppetmaster

Pour un système utilisant DNF (Yum)

Sur le client : yum install puppet
Sur le serveur : yum install puppet-server

Pour démarrer le service

Sur le client : service puppet start
Sur le serveur : service puppetmaster start
